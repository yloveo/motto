格言

格言介绍
格言即是一句有意义的话，调用这句话可以丰富你的网站，使你的网站更加有内涵，同时你不需要有任何网络技术，我们提供现成的代码与数据。
使用方法
格式：https://api.yloveo.com/api.go?id=gy&type=[类型]&kind=[种类]
提示：kind为空将随机输出任意种类的格言。
例： https://api.yloveo.com/api.go?id=gy&type=xml&kind=music
解释：将以XML的格式输出一句歌词。
效果：
<content>曾沿着雪路浪游，为何为好事泪流，谁能凭爱意要富士山私有？</content>
<author>林夕</author>
<source>富士山下</source>
参数一览
请求参数
名称 字段 必须 取值
类型 type 是 text / json / xml
种类 kind 否 poetry / music / movie / philosophy / joke
返回参数（返回text纯文本免看）
名称 字段 解释
内容 content 格言的主要内容
作者 author 格言的作者，或者该语句的主人
来源 source 格言来自某个作品，有可能为NULL
